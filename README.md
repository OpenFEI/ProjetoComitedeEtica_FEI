# Intro

Template latex para o envio de projetos de pesquisa que envolvam seres humanos nos Comitês de Ética em todo o país.

## Como usar

    $ git clone <this repo>
    $ git submodule init
    $ git submodule update

Em seguida, basta editar os arquivos .tex e .bib normalmente.
